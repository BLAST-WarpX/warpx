# 石英边界条件物理验证指南

## 概述

本文档介绍如何通过物理对比验证石英边界条件的有效性。通过比较石英边界条件和PEC边界条件在相同条件下的物理表现，可以验证石英边界条件是否正确实现了介电材料的物理特性。

## 物理原理

### 石英边界条件的物理特性

1. **电场边界条件**:
   - **切向分量**: E_tangential_outside = E_tangential_inside (连续)
   - **法向分量**: D_normal_outside = D_normal_inside (电位移矢量连续)
     - 由于 D = ε₀εᵣE，所以 E_normal_outside = εᵣ × E_normal_inside
     - 对于石英，εᵣ = 3.8

2. **磁场边界条件**:
   - **所有分量**: B_outside = B_inside (连续)
   - 因为石英是非磁性材料，μᵣ ≈ 1

### PEC边界条件的物理特性

1. **电场边界条件**:
   - **切向分量**: E_tangential = 0 (在边界处为零)
   - **法向分量**: E_normal 可以存在

2. **磁场边界条件**:
   - **法向分量**: B_normal = 0 (在边界处为零)
   - **切向分量**: B_tangential 可以存在

### 1. 电场分布差异

**石英边界条件**:
- 电场在边界处应该表现出介电材料的特性
- 法向电场分量应该被石英的介电常数影响
- 电场分布应该比PEC边界更连续

**PEC边界条件**:
- 切向电场分量在边界处应该为零
- 电场分布应该表现出理想导体的特性

### 2. 磁场分布差异

**石英边界条件**:
- 磁场应该保持连续
- 由于石英是非磁性材料，磁场分布应该与真空相似

**PEC边界条件**:
- 法向磁场分量在边界处应该为零
- 磁场分布应该表现出理想导体的特性

### 3. 能量分布差异

**石英边界条件**:
- 电磁能量应该能够部分穿透边界
- 能量分布应该表现出介电材料的特性

**PEC边界条件**:
- 电磁能量应该被完全反射
- 能量分布应该表现出理想导体的特性

## 验证指标

### 1. 电场强度对比

- **石英边界**: 边界处电场强度应该比PEC边界低
- **PEC边界**: 边界处切向电场应该为零

### 2. 电场连续性

- **石英边界**: 电场应该表现出更连续的分布
- **PEC边界**: 电场在边界处应该有明显的跳跃

### 3. 能量反射率

- **石英边界**: 部分能量应该能够穿透边界
- **PEC边界**: 能量应该被完全反射

## 物理验证标准

### ✅ 验证通过的标准

1. **电场法向分量**: 石英边界的法向电场应该被介电常数影响
2. **电场连续性**: 石英边界的电场分布应该比PEC边界更连续
3. **能量穿透**: 石英边界应该允许部分电磁能量穿透
4. **磁场连续性**: 石英边界的磁场应该保持连续

### ❌ 验证失败的标准

1. 石英边界和PEC边界表现出完全相同的物理特性
2. 石英边界的电场分布不符合介电材料的理论预期
3. 磁场在石英边界处不连续
